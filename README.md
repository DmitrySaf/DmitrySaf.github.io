<h2>Installation and launching:</h2>

1. Install the project:

```
git clone https://github.com/DmitrySaf/MetaLampToxin/
```
2. Go to the directory:

```
cd MetaLampToxin
```
3. Run the dev server:

```
npm run dev
```
4. Build the project:

```
npm run build
```
5. Run eslint:

```
npm run eslint
```
